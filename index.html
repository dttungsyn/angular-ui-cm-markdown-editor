Instant Codemirror Markdown Editor

Simple Markdown Editor

Easily create markdown editor with separately designed helping syntax toolbar, auto generate table contents from heading, along with Angular directive with 2 ways data-binding model option!

Git Repository

tar.gz .zip

Features

  • Instant translation from markdown to HTML.

  • Toolbar buttons self designed independently

  • Auto generate Table of Contens

  • Angular directive with 2 ways data binding.

Use with self-designed toolbar

  • Heading 1

  • Heading 2

  • Heading 3

  • Heading 4

In this example, i created a toolbar buttons using bootstrap and font-awesome, pass its selector string in as a config option {toolbarContainer: '.btn-toolbar'}. Inside the toolbar, only buttons that has attribute data-editorbtn make sense to our editor, and its value specify its corresponding function. Ex: <a data-editorbtn="h1"><h1>Heading 1</h1></a>. Visually, you can design the toolbar for yourself, and put it any where you want.

Source

codemirrorOptions = {
  toolbarContainer: '.btn-toolbar'
};
new MyCodemirrorUI( $('#codemirror-markdown-editor-2'), codemirrorOptions );

Here is the list of functions provided:

Item Value
h[1-6] Heading element with the level from 1 to 6
bold make selected text bold
italic make selected text italic
strikethrough make selected text strikethrough
undo step back one step
redo step forward one step
list create list
table create table
eye change view between markdown, preview or both
fullscreen make fullscreen view editor
script create script ( javascript, html, C, java, python,....)

Use as Angular directive

Source

We have to create a javascript file for definition of Angular module and controller, remember to include ui-codemirror-markdown module.

controller.js
angular.module('MyModule', ['ui-codemirror-markdown'])
    .controller('MyController', ['$scope', controller]);

function controller($scope){
    $scope.content = "#This is codemirror";
    $scope.editorOptions = {
        lineNumbers: false
    };
}

Then include javascript file and setup html element

html
<script src="js/controller.js"></script>
<body data-ng-app="MyModule">
    ...
    <section data-ng-controller="MyController">
      <div class="row" ng-model="content" ui-codemirror-markdown="editorOptions"></div>
    </section>
    ...      
</body>
