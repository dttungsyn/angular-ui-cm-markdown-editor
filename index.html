Instant Codemirror Markdown Editor

Simple Markdown Editor

Easily create markdown editor with separately designed helping syntax toolbar, auto generate table contents from heading elements, along with Angular directive with 2 ways data-binding model option!

Read Me

Features

  • Instant translation from markdown to HTML.

  • Toolbar buttons self designed independently

  • Auto generate Table of Contens

  • Angular directive with 2 ways data binding.
